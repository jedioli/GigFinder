GigFinder
=========

A super awesome geographic music trend tool!

Dependancies
=========
Python libraries : pylast

Can be installed with pip install pylast.


Instructions
=========
1. Run genresnoop2.py which will read in the artist id's from artist.txt and output the artist id's with the top 5 genres in artistGenre.txt

2. Run countGenre.py which reads in artistGenre.txt and counts the frequencies of genres and outputs to topGenre.txt

3. Run mapTweetsToGenre.py which takes artistGenre.txt and topGenre.txt as inputs and outputs the top 50 genres in the mapData folder with <genre>MapData.txt as the file names.
